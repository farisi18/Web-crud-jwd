# Web-crud-jwd

Cara Menggunakan:
1. Pastikan Anda Sudah Menginstall Xampp
2. Buatlah database pada phpmyadmin sesuai dengan nama database yang tersedia dalam project
3. Import database dari file project
4. Nyalakan Xampp nya dan buka folder project pada browser sesuai nama folder yang berada dalam htdoc "localhost/namaproject"
